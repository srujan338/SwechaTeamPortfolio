// team working style

How We Work Together

Our process — roughly. We're still figuring it out 😄

🌅
Daily Check-ins
Quick 10-min morning sync on what everyone's working on.
🖥
Weekly Demos
Every Friday we show what we built, even if it's broken!
👬
Pair Programming
Stuck on a bug? We hop on a call and debug together.
🔍
Code Reviews
No direct merges to main — every PR needs at least one review.
📋
Task Boards
We track everything on GitHub Issues. Simple, but effective.
💬
Async-first
Prefer written updates in chat; we respect each other's focus time.
